# The Scottish Highlands - Game Info

## Quick Run Instructions
1.  Navigate to the folder: `c:\Users\User\Documents\AL SCOTTISH GUY`
2.  Double-click `index.html`.
3.  The game will open in your default browser.
4.  **Controls**:
    -   **Arrow Keys**: Move the Bagpiper.
    -   **Space**: Shoot musical notes.
    -   **Alt+F4**: (Joke) "Fun" - actually closes the window, don't do this unless you want to quit!
